# Caculadora de Custo Mensal de um Carro

Comando para rodar o programa no terminal linux ubuntu:

    nasm -f elf64 custoMensalCarro.asm && gcc custoMensalCarro.o -o custoMensalCarro -no-pie && ./custoMensalCarro

Dependencias: 

    GCC compiler e NASM
